# YouTube-Comment-Chris-ft-Davidoo-Power-Bi-Project

## 🎵 YouTube Comments Analysis Dashboard (Chris Brown ft. Davido)
📌 Project Overview

This Power BI project analyzes audience engagement and interaction patterns from YouTube comments on the music video Chris Brown ft. Davido. The dashboard provides insights into comment activity, user engagement, artist popularity, music style references, and commenting behavior.

The objective of this project is to transform raw YouTube comment data into actionable insights using Power BI's data modeling, DAX calculations, and interactive visualizations.

## 🎯 Business Objectives

The dashboard answers the following key questions:

1. How many comments were posted?
2. How many likes were received on comments?
3. What percentage of comments received likes?
4. How many replies were generated from comments?
5. Which artists are most frequently mentioned?
6. What music styles are referenced by viewers?
7. Which viewers post multiple comments?
8. When are viewers most active during the day?
9. How does comment activity vary across weekdays?
10. What are the daily comment trends over time?
